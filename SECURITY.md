# Security Policy

## Supported Versions

Only the latest version of Telethings is actively maintained.

| Version | Supported |
| ------- | --------- |
| latest  | ✅        |
| older   | ❌        |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public GitHub issue.

Instead, please report it by opening a [GitHub Security Advisory](https://github.com/IlyasYOY/telethings/security/advisories/new) so it can be reviewed and patched privately before public disclosure.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You can expect a response within a few days.
